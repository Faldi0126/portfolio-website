Resoria

A resort and hotel finder website

My first project is a resort and hotel finder website designed to make it easy for users to find their ideal vacation destination.
Users can quickly find a hotel/resort that closest to them and give it a review, the user also can add their own hotel/resorts.

This project was created using Vue.js and Pinia for the front-end. Node.js, Express.js, PostgreSQL and Sequelize for the backend and database, I also use Tailwind CSS for styling.

Privator

A personal tutor website where the users can find a tutor to increase their skills

Our final group project is a private mentor for course website that provides students with personalized support and guidance in their academic and non-academic pursuits.
This website connects students with mentors who offer one-on-one support and coaching on a variety of subjects.

This web app helps you find a private mentor to support your learning, you can find the instructor based on the location, reviews or category. We also offer on site learning or online with video call features.
This website is built using React.js, Redux, Tailwind CSS, Express.js, PostgreSQL and few third party API.

Sushi Restaurant Website

A Sushi restaurant website built with React and equipped with a comprehensive CRUD feature. The goal of this project was to provide an easy and seamless way for users to browse and add the items they want.
